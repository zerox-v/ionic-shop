## 简介
### 基于Taro、React编写的微信、支付宝小程序，项目正在一点点完善。
## 开始使用
#### 全局安装 Taro
`yarn global add @tarojs/cli`
#### 还原包
`yarn`
#### 运行
`yarn dev:weapp`
#### 打包
`yarn build:weapp`
